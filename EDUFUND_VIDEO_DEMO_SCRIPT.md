## Production Recommendations

### **Technical Setup**

1. **Screen Recording Software**
   - OBS Studio (free) or Camtasia
   - Record at 1920x1080, 60fps
   - Use browser in incognito mode with demo data

2. **Demo Environment**
   - Pre-seed database with realistic demo data
   - Use Stellar testnet accounts with fake XLM
   - Prepare multiple user accounts (student, donor, school, admin)
   - Disable 2FA for smoother demo flow

3. **Editing Software**
   - Adobe Premiere Pro / DaVinci Resolve / Final Cut Pro
   - Add smooth transitions between scenes
   - Include zoom effects on important elements
   - Add background music (inspiring, corporate)

### **Visual Enhancements**

1. **Animations**
   - Use cursor highlighting (circle or glow effect)
   - Add click animations to show user interactions
   - Smooth scroll animations between sections
   - Loading state animations for blockchain transactions

2. **Graphics & Overlays**
   - Lower thirds for user roles (Student, Donor, School, Admin)
   - Progress bars and statistics callouts
   - Blockchain transaction flow diagrams
   - Icon overlays for key actions (wallet, verification, approval)

3. **Color Scheme**
   - Primary: #2563EB (blue)
   - Success: #10B981 (green)
   - Background: #FAFAFA (light gray)
   - Text: #171717 (near black)

### **Text Overlay Guidelines**

1. **Font:** Clean, sans-serif (Inter, Roboto, or similar)
2. **Size:** Minimum 24pt for readability on mobile
3. **Color:** White text with dark shadow or dark text with light background
4. **Position:** Lower third or top center, avoid covering important UI elements
5. **Duration:** Display each text overlay for 2-3 seconds
6. **Animation:** Fade in/out smoothly

### **Timing Breakdown**

| Scene | Duration | Key Message |
|-------|----------|-------------|
| Opening (Landing Page) | 15s | Platform introduction & features |
| Student Journey | 30s | Creating & tracking funding |
| Donor Experience | 30s | Browsing & donating |
| School/Admin | 30s | Verification & monitoring |
| Blockchain | 15s | Transparency & security |
| Closing | 5s | Call to action |

### **Accessibility**

1. Add closed captions/subtitles with key information
2. High contrast visuals
3. Clear, readable fonts (minimum 24pt for on-screen text)
4. Ensure all text overlays are large enough to read

### **Distribution**

1. **YouTube:** Full 2-minute version + 30-second teaser
2. **Website:** Embed on homepage
3. **Social Media:** 60-second and 15-second cuts for Instagram/TikTok/LinkedIn
4. **Pitch Decks:** Include in investor presentations
5. **Onboarding:** Use for new user tutorials

## Alternative Demo Flow (If Time-Constrained)

If you need a faster production, use this simplified flow:

**0-15s:** Landing page tour (hero, features, campaigns, CTA)
**15-45s:** Student creates request and tracks milestones
**45-75s:** Donor browses and donates (show blockchain transaction)
**75-105s:** School approves and verifies milestone
**105-120s:** Blockchain transparency + Admin dashboard
**120-125s:** Closing CTA on landing page

This flow focuses on the complete lifecycle of a single campaign from creation to fund disbursement.

**Note:** Since there's no voiceover, use larger text overlays and more visual cues to guide viewers through the narrative.
